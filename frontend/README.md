# Frontend (React + Vite) - Minimal Scaffold

Quick start

```powershell
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173` (Vite default). The frontend expects the backend at `http://localhost:4000` by default; change `VITE_API_BASE` in `.env` if different.

Pages implemented:
- `/login` — login page
- `/dashboard` — employee dashboard (check-in, check-out, recent history)
